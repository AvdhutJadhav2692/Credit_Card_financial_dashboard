# Credit Card Transaction Report - Power BI Dashboard
## 📋 Project Overview
This project involves the creation of a comprehensive **Credit Card Transaction Report** using Power BI. The objective is to provide real-time insights into key performance indicators (KPIs) such as revenue, transaction volume, and interest earned, while segmenting the data by customer demographics, card categories, and expenditure types.

This dashboard helps stakeholders monitor financial health, track customer behavior, and optimize marketing strategies for different card tiers.

## 🚀 Key Features
*   **KPI Tracking:** Real-time visibility into Total Revenue ($55M), Total Interest ($45M), Transaction Amount ($7.8M), and Total Transaction Count (656K).
*   **Quarterly Analysis:** A combined bar and line chart showing revenue trends and transaction counts across Q1-Q4.
*   **Customer Segmentation:** Analysis of revenue based on:
    *   **Education Level:** (Graduate, High School, etc.)
    *   **Customer Job:** (Business, White-collar, Self-employed, etc.)
    *   **Card Category:** (Blue, Silver, Gold, Platinum)
*   **Transaction Insights:** Breakdown of revenue by expenditure type (Bills, Entertainment, Fuel, etc.) and transaction method (Swipe, Chip, Online).
*   **Financial Metrics:** Detailed table showing revenue, transaction amounts, and interest earned per card category.
*   **Interactive Slicers:** Filter data by Quarter, Gender, Income Level, and Week Start Date for granular analysis.

## 📊 Key Insights from the Data
*   **Top Performer:** The **Blue Card** category is the primary driver of revenue ($46M) and has the highest customer acquisition cost.
*   **Spending Behavior:** The highest revenue is generated through **Bills** and **Entertainment** expenditures.
*   **Demographics:** Customers with a **Graduate** level of education and those in **Business** roles contribute the most to total revenue.
*   **Transaction Method:** **Swipe** transactions are the most popular, accounting for $35M in revenue.
*   **Seasonality:** Revenue peaked in **Q3** ($14.2M) alongside the highest transaction volume (166.6K).
